## Objetivo
---
Crear una plataforma para que los estudiantes puedan solicitar una beca universitaria. Una que sea intuitiva, sencilla y cumpla con todos los requerimientos que se necesiten, para que tanto los estudiantes como el Ministerio esten cómodos con el programa. 

## Duración: 6 meses.
